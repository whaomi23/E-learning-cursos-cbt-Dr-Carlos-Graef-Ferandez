¿Que es HTML?


HTML es lenguaje de Hyper Texto que nos permite desarrollar paginas web a travez de un programa que manege la estructura de HTML para comnezar a desarrolar nuestra propia pagina web sin tener que pagar hacer como nosotros queramos crear la pagina web que utilzaremos para brindar informacion de caulquier tipo.

HTML es lenjuaje mas facil de aprender ya que no ser requiere de muchos conocimientos de informatica para crear un sitio web simple ya que se puede decir que muchas mas personas puden aprenderlo muy rapido.
